# Utilitarianism-Game

##Overview
**Utilitarianism** is an interactive ethical decision-making game developed for the **Future Business Leaders
of America State Leader Conference 2025 Computer Game & Simulation Programming**. The project was recognized as
Finalist, where it competed with 12 other teams from across the state.

**Utilitarianism** confront players with different moral dilemmas such as The Trolly Problem, and every decision carries
weight. Players are presented with difficult scenarios where they must choose whether or not to save a person. Each decision
influences the gameplay and the outcome of the game.

##Development
This project was collaborated with another individual using:
- **Game Engine**: Unity Game Engine
- **Programming Language**: C#
- **Game Design Concepts**: Custom Animations, Voice Lines, Graphics and UI Designs, Interactive and Immersive gameplay.

##Core Gameplay Mechanics
This project prioritized player experience with enhanced UI/UX design and mechanics, including:
- Event-driven programming (decision-making)
- Random Number Generator (RNG)
- Interaction with NPCs (Voice-lined and dialogues)
- Unique level design (Each level is completely different from the other)
